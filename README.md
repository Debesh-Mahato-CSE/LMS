# 📚 Library Management System

A full-stack Library Management System designed to efficiently manage books, users, and borrowing operations. It also includes modern AI features like a chatbot and book recommendation system.

---

## 🚀 Features

- 📖 Add, update, and delete books  
- 👤 User management (Admin / Librarian)  
- 🔍 Search books by title, author, or category  
- 📊 Track available and issued books  
- 🔐 Secure login system (with password encryption)  
- 📅 Issue and return book system  
- 🤖 AI Chatbot support  
- 📚 AI-based book recommendation system  

---

## 🛠️ Tech Stack

### Frontend:
- React.js  
- HTML5  
- CSS3 / Tailwind CSS  

### Backend:
- PHP (REST API)  

### Database:
- MySQL
